Desarrolladora de software en formación (Grado Superior DAM), donde consolido competencias técnicas en arquitectura de sistemas, programación orientada a objetos (POO) y gestión de bases de datos. 

Mi valor diferencial técnico y metodológico reside en una sólida trayectoria profesional previa, en la que puedo destacar

Rigurosa capacidad analítica, precisión técnica y un pensamiento lógico-espacial, elementos que resultan indispensables para la resolución metódica de problemas complejos y el diseño arquitectónico de software.
Habilidades directivas consolidadas como gestión eficiente de equipos, la optimización operativa, la toma de decisiones bajo presión y una estricta orientación a la consecución de objetivos (KPIs).

Esta conjunción interdisciplinar me capacita para abordar el ciclo de vida del software no solo desde la ejecución técnica del código, sino con una visión integral, altamente organizativa y orientada a la eficiencia empresarial.
